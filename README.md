# todoCartoes
Projeto  sendo desenvolvido em  Vue no front-end e Graphql no back-end. 
Ainda refatorando alguns componentes para melhor organização do projeto.



Sobre a instalação:

Front-end

npm install

cd/pasta do projeto

npm run serve

Back-end

npm install

cd/pasta do projeto

npm start
